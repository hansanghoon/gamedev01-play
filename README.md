## 게임 목록

- 🪐 스티커 플래닛 (첫 번째 앱! — 습관을 완료하면 도트 스티커를 뽑아 모으는 습관 기록 앱.
  폰에서 "홈 화면에 추가"하면 진짜 앱처럼 설치되고, 기록은 각자 폰에만 저장돼요)
- ⏱️ 타이머 (컵라면·계란 등 기본 타이머 + 시·분·초로 직접 만드는 타이머.
  기다리는 동안 도트 장면이 나오고, 50분 이상은 "집중의 숲"에서 나무가 자라요. 만든 타이머는 각자 기기에 저장돼요)
- 점프 게임
- Faded Memory (추억 모으기 탐험)
- 꿈속 탐험가 (카드 로그라이크)
- 섬 디펜스 (고양이와 대포)
- 스파이더 액션 (도시 스윙)
- 세계 탐험
- 🔐 방탈출 시리즈 (첫 3D 게임! — 유령의 도서관 / 어두운 아파트 / 괴물의 복도)

모두 하나의 HTML 파일로 되어 있어서, 설치 없이 브라우저로 열면 바로 실행됩니다.
(3D 방탈출은 Three.js 라이브러리를 인터넷에서 불러오기 때문에 온라인 상태에서 플레이해 주세요.)
